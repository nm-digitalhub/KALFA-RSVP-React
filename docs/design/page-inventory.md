# KALFA — Page Inventory

> Full inventory of every App-Router route, classified by area, shell, and render type.
> Generated by a design-system audit team (2026-07-08). Companion docs: `page-design-briefs.md`,
> `ui-component-inventory.md`, `reusable-components-plan.md`, `responsive-rtl-audit.md`, `design-audit-summary.md`.

## How to read this

- **Type** — `page.tsx`/`layout.tsx` are **Server Components** across the whole app (verified: not one carries `'use client'`). Interactivity lives in co-located `*-form.tsx` / `*-client.tsx` Client Components that the server page composes.
- **Shell** — the layout chrome a route renders inside:
  - **Root** — `src/app/layout.tsx` only (`<html lang="he" dir="rtl">`, Heebo font). No sidebar. Used by Public + Auth.
  - **AppShell** — `src/components/app-shell.tsx` via `(customer)/app/layout.tsx`. RTL sidebar (`side="right"`), offcanvas mobile drawer, sticky top bar (search placeholder + OrgSwitcher + account menu), `max-w-5xl` content container.
  - **AdminShell** — `src/components/admin-shell.tsx` via `(admin)/admin/layout.tsx`. Same pattern, longer nav, no search/org-switcher.
- **Route params** — `[id]` = event id, `[campaignId]` = campaign id, `[guestId]` = guest id, `[token]` = opaque public token.

## 1. Public — root layout, no shell

Guest-facing and marketing surfaces. Mobile-first; each page owns its full layout.

| Route | File | Type | Shell | Purpose |
|---|---|---|---|---|
| `/` | `(public)/page.tsx` | Server | Root | Marketing landing — hero, feature sections, CTA to `/app` (or `/auth`). |
| `/privacy` | `(public)/privacy/page.tsx` | Server | Root | Privacy policy (legal, static; shares `_legal.tsx` layout). |
| `/terms` | `(public)/terms/page.tsx` | Server | Root | Terms of service (legal, static; shares `_legal.tsx` layout). |
| `/join/[token]` | `(public)/join/[token]/page.tsx` | Server | Root | Organization invite acceptance (token-gated). |
| `/r/[token]` | `(public)/r/[token]/page.tsx` | Server | Root | **Public RSVP** — the guest-facing response surface. Token-gated to one guest + one event. Highest-traffic, most security-sensitive page. |

_Shared:_ `(public)/_legal.tsx` (legal-page wrapper). Route handler: `(public)/g` (no page).
